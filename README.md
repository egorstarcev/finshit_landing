# 🛡 ФинЩит (FinShield)

<p align="center">
  <b>Твоя финансовая защита</b><br>
  Цифровая платформа против финансового мошенничества
</p>

---

## 🚀 О проекте

**ФинЩит** — это мобильное приложение, которое обучает пользователей распознавать мошеннические схемы и принимать безопасные решения в реальных ситуациях.

Проект сочетает:
- обучение  
- симуляции  
- персонализацию  

---

## 🎯 Проблема

Финансовое мошенничество в России ежегодно наносит ущерб более **20 млрд рублей**.

Особенно уязвимы:
- мигранты  
- пенсионеры  
- студенты  

---

## 💡 Решение

ФинЩит предлагает интерактивный подход:

- 📞 симуляция мошеннических звонков  
- 🎮 обучение через сценарии  
- 🌍 мультиязычный интерфейс  
- 🧠 адаптация под пользователя  

---
